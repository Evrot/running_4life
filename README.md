Running 4Life — Running Companion SPA
Description

Running 4Life is a React single-page application that allows users to log their runs and visualize routes on an interactive map. The app centralizes workout tracking and route planning in one simple dashboard designed for recreational runners.

Features

Add and view running workouts (date, distance, duration, notes)

Interactive route visualization using Mapbox

Individual run details page

Responsive mobile-first layout

Client-side state management using React Context

Basic unit testing with Vitest and React Testing Library

Technologies Used

React (Vite)

React Router

React Context API

Mapbox GL JS + Directions API

Vitest + React Testing Library

CSS (mobile-first responsive design)

Vercel (deployment)

Installation
npm install
npm run dev
Environment Variables

Create a .env.local file in the project root and add:

VITE_MAPBOX_TOKEN=your_mapbox_token_here

You can obtain a free token from Mapbox.

Running Tests
npm test
